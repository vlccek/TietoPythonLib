Vlans¶

  • everyting about vlans is in module vlan

  • for adding vlans to switch you can use class Vlans.

  • you can create vlan in vlans by method Vlans.add_by_params() or add_by_dict()

  • Class Vlans can be also displayed as table by simply printing it

  • You can also delete vlan from Vlans class by method delete()

  • Methods Vlans.delete(), Vlans.add_by_params() and Vlans.add_by_dict() also sorts vlans in Vlans object by their id after addition/deletion

Adding vlans by Vlans.add_by_params()¶

my_sweat_vlans = Vlans()
my_sweat_vlans.add_by_params(id=10)
my_sweat_vlans.add_by_params(20, type="public", auto_vxlan=False, description="test2", stats=True, active=True, untagged_ports=[243, 244, 245], active_ports=[666], ports=[243, 244, 245, 666])
print(my_sweat_vlans)

Adding vlans by Vlans.add_by_dict() method¶

my_sweat_vlans = Vlans()
vlan_template = {
         "id": 1,
         "type": "private",
         "auto-vxlan": False,
         "replicators": None,
         "scope": "local",
         "description": "",
         "active": False,
         "stats": False,
         "ports": [],
         "untagged_ports": [],
         "active_ports": [],
         "vxlan": None,
         "vxlanmodule": ""
     }
   my_sweat_vlans.add_by_dict(vlan_template)
