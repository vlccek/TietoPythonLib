Diags¶

Module to handle downloading diags from switches without need of root privilages.

diags.download_diags(hostname: str = 'switch-ip', port: int = 22, timeout: int = 60, keepalive: int = 60) → None¶

Downloads diags from switch on <hostname> (from settings.py on default).

Parameters

  • hostname – hostname that we connect and download from, by default its taken from settings.py

  • port – SSH port, by default its taken from settings.py

  • timeout – timeout, defaults to 60

  • keepalive – keepalive, defaults to 60

diags.download_diags_from_switches(hostnames: List[str])¶

Downloads diags from switches in <hostnames> list.

Parameters

hostnames – hostnames that we connect and download from, by default its taken from settings.py

diags.send_command_with_login(connection, command)¶

Sends command that needs login.

Parameters

  • connection – SSH Paramiko Client

  • command – Command that is sent on switch.
