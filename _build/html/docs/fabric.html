class fabric.Fabric(hostname: str, username: str, password: str, port: int = 22, timeout: int = 60, keepalive: int = 60)¶

Created object that iminiadly connect to that switch and find all switch in fabric

pepa = Fabric("pepa.cz", "root", "pa$$word123")
Parameters

  • hostname – hostname that we connect

  • username – username of user that we connect

  • password – password to switch

  • port – port, defaults to 22

  • timeout – timeout, defaults to 60

  • keepalive – keepalive, defaults to 60
